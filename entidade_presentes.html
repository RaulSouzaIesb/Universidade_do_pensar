Cadastro de Entidades

Salas de Aula

O sistema permite o cadastro de salas de aula, incluindo informações como prédio, número da sala e capacidade.

Restrições de Integridade: O número da sala deve ser único dentro de cada prédio.

Departamentos

O sistema permite cadastrar departamentos, especificando o nome do departamento, o prédio associado e o orçamento.

Restrições de Integridade: O nome do departamento deve ser único.

Cursos

O sistema permite o cadastro de cursos, com informações como código do curso, título, departamento responsável e número de créditos.

Restrições de Integridade: O código do curso deve ser único.

Instrutores

O sistema permite cadastrar instrutores, incluindo ID, nome, departamento e salário.

Restrições de Integridade: O ID do instrutor deve ser único.

Seções de Curso

Permite o registro de seções de cursos, especificando o curso, ID da seção, semestre, ano, prédio, número da sala e identificador do horário.

Restrições de Integridade: A combinação de curso, ID da seção, semestre e ano deve ser única.

Alunos

Permite cadastrar alunos, fornecendo informações como ID, nome, departamento e créditos acumulados.

Restrições de Integridade: O ID do aluno deve ser único.

Inscrições em Cursos

O sistema permite o registro de inscrições em cursos pelos alunos, incluindo ID do aluno, código do curso, ID da seção, semestre, ano e nota.

Restrições de Integridade: A combinação de aluno, curso, ID da seção, semestre e ano deve ser única.
